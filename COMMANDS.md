# Commands

This is list of all the availabe commands by typing `command + "what you want to search"` in the search bar.

| Command      | Corresponding Sites          |
| ------------- |:-------------:|
| ?      | HELP |
| a:      | AMAZON |
| d:      | GOOGLE DRIVE |
| G:       | FACEBOOK |
| g:       | GITHUB |
| i:       | INSTAGRAM |
| m:       | MYANIMELIST |
| r:      | REDDIT |
| t:      | TWITTER |
| tw:    | TWITCH  |
| w:      | WIKIPEDIA  |  
| y:      | YOUTUBE  |
